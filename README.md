# TASKY

## Goal
A CLI app for managing tasks. 

### Install with
```python
pip install TaskyCLI_RunjeetNarula
```

### Example Usage
```python
tasky --help
```

## Commands
- add <description>
- update <id>
- delete <id>
- mark-in-progress <id>
- mark-done <id>
- list [OPTIONS]
    --done
    --todo
    --in-progress
